## 🧑‍💻 Tech Stack

- 🖥️ **Frontend**: ReactJS + TailwindCSS
- 🛠️ **Backend**: NodeJS
- 🗄️ **Database**: MySQL
- ☁️ **Media Storage**: Cloudinary (images)
